# tiny-paws-market
ペットショップサイト（架空のショップサイト）

## サイト概要
自主制作の架空ペットショップサイトです。
TOP/GUIDE/ABOUT/CONTACTを想定し、共通ヘッダー・フッターを使いながら、
各ページでペット一覧やガイド、問い合わせファームなどをレイアウトしました。

## アピールポイント
- SP：ハンバーガー/PC：横並びのグローバルナビ
- ペット一覧・ガイド・こだわり紹介をFlexbox/CSS Gridでカード表示
- SPは１カムラ、PCは２～３カムラに変化するレスポンシブレイアウト
- CONTACTフォームのラベル/必須表示やフォーカス時の装飾で、入力のしやすさを意識
